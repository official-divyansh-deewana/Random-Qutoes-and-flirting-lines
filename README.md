## 🛠 How To Use

You can use the following endpoints to get data in a beautiful JSON format:

| Type | Endpoint | Description |
| :--- | :--- | :--- |
| **English Quote** | `/?random` | Get a random motivational quote |
| **Hinglish Flirt** | `/?random-Flirt` | Get a random flirting line in Hinglish |
